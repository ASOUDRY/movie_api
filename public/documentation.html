Kaiju Catalogue This is a program designed to pull up a catalogue of Kaiju movies, and allow users to pull information from this movies at a simple command. They can also make a list of their own favorite kaiju movies as well as to add their own suggestions for movies on the Api. This Get Request fetches all the movies from the movie collection. This is used to fetch all the movies for the initial movie-list app.get('/movies', passport.authenticate('jwt', { session: false }), (req, res) => { Movies.find() .then((movies) => { res.status(201).json(movies); }) .catch((err) => { console.error(err); res.status(500).send('Error: ' + err); }); }); This Get Request fetches a single movie using the Title parameter, which is used to fetch a single movie. app.get('/movies/:Title', passport.authenticate('jwt', {session: false}), (req, res) => { Movies.findOne({ Title: req.params.Title }) .then((title) => { res.json(title); }) .catch((err) => { console.error(err); res.status(500).send('Error: ' + err); }); }); app.get('/movies/Id/:Id', passport.authenticate('jwt', {session: false}), (req, res) => { Movies.find({ _id: req.params.Id }) .then((movie) => { res.json(movie);
