Kaiju Catalogue This is a program designed to pull up a catalogue of Kaiju movies, and allow users to pull information from this movies at a simple command. They can also make a list of their own favorite kaiju movies as well as to add their own suggestions for movies on the Api. Fetch Movies Returns a string from from the the directory movies. In the future it will return a Json Object with a list of all the movies stored in the directory. The endpoint that it uses is /movies. The HTTP request type is a GET Request. There are no parameters for this request type. Example Request app.get('/movies', (req, res) => { res.send("All The Movies"); }); Example Output "All the movies." Future Example Request app.get('/movies', (req, res) => { res.json(Catalogue); }); Future Example Output { { "Name": "American Werewolf in London", "Genre": "Horror/Comedy", "Director": "John Landis" }, { "Name": "The Howling", "Genre": "Horror", "Director": "Joe Dante" }, } Fetch Movie Returns a string from a subdirectory specified by its name. In the future it will return a Json Object by name that contains the name, director, lead actora and synopsis of the movie. The end point is /:Genre/:Title. The HTTP request type is a GET type request. There are two paremeters,[Genre], and [Title]. Both parameters are sub directories which need to be specified in the Url. The parameter for this request is req.body. The req.body requires a object that contains the .name property. A object that lacks that property triggers the if statement. [Account] specifies the subdirectory that this request accesses, while req.body contains json object that modifies the existing json object.
